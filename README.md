Tabuada interativa

Uma tabuada onde o usuário escolhe dois números e o sinal de operação

Funcionalidades
- Escolha entre soma, subtração, multiplicação ou divisão
- Exibe o resultado de dois números
- Não divide por zero

Como Usar
1. Execute o arquivo 'tabuada interativa.py'
2. Digite o numero desejado
3. Escolha a operação (1, 2, 3 ou 4)

Tecnologias usadas
- Python 3

Objetivo
Treinar condicionais, loops e entrada de usuário
